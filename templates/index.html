WoW data folder setup

⚠ WoW must not be installed in Program Files. Chrome's folder picker cannot write there. Install WoW to a user-writable path such as C:\Games\World of Warcraft before continuing.

Once WoW is in a user-writable location, setup is straightforward — no junctions or admin required.

Step 1 — create the data folder (skip if it already exists)

New-Item -ItemType Directory -Path "<WoW>\Interface\AddOns\Simdragosa\data" -Force

Replace <WoW> with your install path, e.g. C:\Games\World of Warcraft\_retail_

Step 2 — connect the folder here

Click Connect folder and select the data folder at <WoW>\Interface\AddOns\Simdragosa\data. Done — SimdragosaData.lua will be written there after every sim and WoW loads it on /reload.

How to find your Raidbots session ID

  1. Open raidbots.com in Chrome or Edge (no account required).
  2. Press F12 to open DevTools.
  3. Go to the Application tab.
  4. In the left sidebar, expand Cookies and click https://www.raidbots.com.
  5. Find the cookie named raidsid and copy its Value.
  6. Paste it in the field above and click Save.

The raidsid cookie acts as your session token. It does not require a paid Raidbots account, but a free account increases your sim quota.

Welcome to Simdragosa 👋

To submit sims you need a Raidbots session ID. This is a cookie from raidbots.com that lets the app submit sims on your behalf — no paid account needed.

  1. Open raidbots.com in this browser.
  2. Press F12 → Application tab → Cookies → raidbots.com
  3. Copy the value of the raidsid cookie.
  4. Paste it in Settings → Raidbots Session ID.

You can do this after closing this guide — the setting is always accessible in the top-right Settings panel.

Add your character

Click + Add Character in the sidebar and paste your SimC string. The app will auto-fill your name, spec, realm, and region.

How to get your SimC string:

  • Install the SimulationCraft in-game addon and use /simc, or
  • Go to raidbots.com → Top Gear → Get SimC button at the top.

You can add multiple characters and multiple specs per character.

Auto Lua delivery (optional)

If you use the Simdragosa WoW addon, you can have SimdragosaData.lua written to your WoW folder automatically after every sim — no manual file moving.

  1. In Settings, enable Auto-deliver Lua after sims.
  2. Click Connect folder and select your Documents\Simdragosa folder.
  3. If this is your first time, click the ? next to "WoW data folder" for the one-time junction setup instructions.

This requires Chrome or Edge. You can skip this step and use the manual download button instead.
